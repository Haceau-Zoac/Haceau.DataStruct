介绍
---
Haceau.DataStruct是一个使用C#实现的数据结构类库。开源协议为MIT。

类
---
* LinkedList -- 链表类
* DynamicArray -- 动态数组类
* Stack -- 栈类

使用
---
1. 导入dll文件（在Release里）
2. 添加using Haceau.DataStruct
3. 使用LinkedList（链表）、DynamicArray（动态数组）和Stack（栈）吧！
